### **スライド5：2.1 データの分析と前処理**

- **データ概要**
  - 合計30枚の画像
    - OKワーク3つ × 各5枚（異なる照明位置・明度）
    - NGワーク3つ × 各5枚（異なる照明位置・明度）
  - NGは人の目でも確認可能な小さな鋳巣
- **前処理の検討**
  - **ワーク部分の抽出**
    - 背景やつなぎ目を除去し、ワークのみを対象とする
  - **照明条件の正規化**
    - 明度やコントラストの補正で照明の影響を軽減
  - **データ拡張**
    - 回転、反転、明度調整などでデータ量を増加

*配置：データ概要と前処理の検討を見出しで分け、箇条書きで内容を記載。可能であれば、簡単な図やイメージを添付。*

### **スライド6：2.2 モデルの選定と構築**

- **モデル選定のポイント**
  - **データ数が少ないため、転移学習の活用を検討**
  - **異常検知モデルの検討**
    - オートエンコーダー、One-Class SVMなど
- **構築手法**
  - 既存の深層学習モデルを微調整（Fine-tuning）
  - データ拡張で学習データを補強

*配置：モデル選定のポイントと構築手法を見出しで分け、箇条書きで内容を記載。専門用語には簡単な説明を添えると親切です。*

### **スライド7：2.3 欠陥の特徴抽出**

- **鋳巣の特徴**
  - 不規則な形状、小さなサイズ
- **特徴抽出手法**
  - **エッジ検出**
    - 画像の輪郭を強調し、欠陥部分を抽出
  - **テクスチャ解析**
    - 表面の質感の違いを検出
- **他の欠陥との識別**
  - ツールマーク：規則的な線状
  - 水滴：円形に近い形状

*配置：鋳巣の特徴と特徴抽出手法、他の欠陥との識別を分けて記載。視覚的な理解を助けるために、図やイラストを追加すると良いです。*

### **スライド8：2.4 欠陥位置の可視化**

- **可視化の目的**
  - モデルが検出した欠陥の位置を明確に示す
  - 検出結果を直感的に理解しやすくする
- **可視化手法**
  - **Grad-CAM（グラッドカム）**
    - モデルが注目している領域をヒートマップで表示
  - **セグメンテーションモデル**
    - 欠陥部分をマスクとして表示

*配置：可視化の目的と手法を見出しで分け、内容を記載。Grad-CAMやセグメンテーションの簡単な説明を添えると理解が深まります。*

---

### **スライド9：2.5 モデルの評価と精度向上**

- **評価指標**
  - **適合率（Precision）**：正しく検出した欠陥の割合
  - **再現率（Recall）**：見逃さずに検出できた割合
  - **F1スコア**：適合率と再現率の調和平均
- **評価方法**
  - **クロスバリデーション**
    - データを分割して学習と評価を繰り返し、信頼性を高める
- **精度向上の手法**
  - ハイパーパラメータの最適化
  - 過学習の防止（ドロップアウト、正則化）

*配置：評価指標、評価方法、精度向上の手法を見出しで分け、簡潔に記載。専門用語には補足説明を加えると親切です。*
